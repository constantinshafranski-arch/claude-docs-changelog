Release Notes

Key Highlights

  • Claude Haiku 3 retired — claude-3-haiku-20240307 was officially retired on April 20, 2026; all requests now return an error — migrate to Claude Haiku 4.5.
  • workspace_restricted_developer role — New "workspace_restricted_developer" role added to workspace member create, retrieve, list, and update endpoints across all relevant Admin API docs.
  • Admin API curl examples — curl request examples added to all 35 Admin API reference pages covering organizations, invites, users, API keys, workspaces, members, cost reports, and usage reports.
  • SSH pre-configuration for teams — Administrators can now distribute SSH connections to team members via sshConfigs in managed settings; connections appear as read-only in each user's environment dropdown.
  • Hooks system improvements — SessionStart hook added alongside CwdChanged for direnv integration; agent hooks marked experimental; if field now matches against Bash subcommands and removed PermissionDenied from supported tool events.
  • Terminal configuration guide overhauled — The terminal setup doc was restructured into symptom-driven sections covering Shift+Enter, Option key shortcuts, notifications, tmux, and Vim mode, with expanded terminal compatibility tables.
