Platform

Key Highlights

  • Claude Fable 5 &amp; Mythos 5 model support — Both new models added to code execution, programmatic tool calling, web search/fetch dynamic filtering, structured outputs, and tool search across the platform.
  • Advisor tool: mid-conversation nudge &amp; Haiku/Opus system prompts — New guidance for injecting a plain-text nudge to increase advisor call rates on Haiku executors (~7 pp improvement), plus an alternative Haiku coding system prompt and an Opus under-calling checkpoint.
  • Vision: revised token cost model &amp; coordinate workflow — Image cost now calculated as 28×28 px visual token patches; added full resized_size reference implementation, bounding-box coordinate workflows, and updated cost tables.
  • ZDR / data retention: per-workspace override for Fable 5 &amp; Mythos 5 — Organizations with a ZDR arrangement can now enable 30-day retention on individual workspaces via Console &gt; Settings &gt; Workspaces to unlock Fable 5 and Mythos 5 in those workspaces.
  • Refusals &amp; fallback: new frontier_llm refusal category — Added "frontier_llm" as a named stop_details.category for requests that could assist competing AI model development.
  • Claude Code: claude-plugins-official marketplace registration clarified &amp; plugin submission URL updated — Official marketplace now registers on first interactive launch (not automatically); non-interactive scripts must add it explicitly. Plugin submission for claude.ai now requires Team/Enterprise and uses a new admin-settings URL.
