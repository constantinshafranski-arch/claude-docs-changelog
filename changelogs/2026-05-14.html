Key Highlights

  • Agent SDK credit change (June 15, 2026) — Agent SDK and claude -p usage on subscription plans will draw from a new separate monthly credit starting June 15, 2026; updated in the headless mode and legal docs.
  • Model deprecation cleanup — Claude Sonnet 3.7, Claude Haiku 3.5 (retired on API), Claude Opus 3, Sonnet 3, and Haiku 3 rows removed from pricing and compatibility tables across batch processing, prompt caching, tool use, and search results docs.
  • Bedrock &amp; Vertex AI lifecycle schedules — Both platform pages now include a disclaimer that Bedrock/Vertex retirement dates are set independently by AWS/Google Cloud and may differ from Anthropic's own schedule; retirement dates updated for Claude Sonnet 3.7 (Bedrock: retired Apr 28; Vertex: retired May 11) and Claude Opus 4 (Bedrock: retiring May 31).

  • Extended thinking model comparison table — Claude Sonnet 3.7 row removed; Claude Opus 4.7 thinking output now documented as omitted by default (set display: "summarized" to receive it), matching Claude Mythos Preview behavior.
  • Output styles doc restructured — "How output styles work" and "Comparisons to related features" sections rewritten with a new comparison table covering Output styles, CLAUDE.md, --append-system-prompt, Agents, and Skills; keep-coding-instructions frontmatter field now explained with a worked example.
