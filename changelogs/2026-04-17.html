Key Highlights

  • claude-opus-4-7 model — New frontier model added to the API as the top-tier option for long-running agents and coding tasks, now appearing as the first entry in BetaManagedAgentsModel across all agent and session endpoints.
  • user-profiles-2026-03-24 beta flag — New beta header value added to AnthropicBeta across all beta API endpoints (now 20 union members instead of 19), accompanied by 6 new CLI user_profiles sub-command docs.
  • BetaCompactionBlock encrypted_content field — New optional encrypted_content string added to BetaCompactionBlockParam and BetaCompactionBlock for round-tripping opaque compaction metadata, affecting Messages create/count_tokens and batch endpoints.
  • BetaModelInfo xhigh capability — New xhigh extended thinking capability field added to BetaCapabilitySupport in the Models API list and retrieve endpoints.
  • Beta headers cURL label fix — Code example shell tab label corrected from "Shell" to "cURL" in the beta-headers documentation.
  • 196 API reference docs updated, 6 new CLI user_profile docs added — Broad sweep updating model lists, beta header enumerations, and compaction block schemas across all SDK language variants (Python, TypeScript, Go, Java, Ruby, C#, CLI).
