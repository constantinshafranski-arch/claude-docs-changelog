Key Highlights

  • PHP SDK added — Full PHP client library launched with beta endpoints for Agents, Environments, Files, Memory Stores, Sessions, Skills, User Profiles, Vaults, Messages, Models, and Webhooks — 131 new docs across all API surfaces.
  • Response examples added to all Admin API endpoints — Every endpoint (API Keys, Cost Report, Invites, MCP Tunnels, Organizations, Rate Limits, Usage Report, Users, Workspaces) now includes a concrete JSON response body example.
  • Endpoint naming standardized across Admin API — Generic section headings like "Retrieve", "List", and "Create" replaced with descriptive names (e.g., "Get API Key", "List Organization Rate Limits", "Create Tunnel Certificate") for improved discoverability.
  • Return type syntax corrected across entire API reference — Type annotations changed from assignment-style (APIKey = object {...}) to plain object notation (APIKey object {...}) across hundreds of endpoints covering all 8 SDK languages.
  • MCP Tunnel Certificate management docs updated — All CRUD operations for tunnel certificates now include response examples with fingerprint, archived_at, and expires_at fields.
  • Volume note — This release touched 357 updated docs and 131 new PHP SDK docs; many individual per-language endpoint entries were updated in bulk and are omitted from per-entry detail above.
