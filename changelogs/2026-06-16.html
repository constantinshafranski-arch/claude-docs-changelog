Claude Code settings

Updated

Six new settings added for Claude Code v2.1.119+ covering push notifications, auto-compact, file checkpointing, theme, and verbose output; legacy ~/.claude.json note updated to reflect the new set of keys moved to settings.json.

  • agentPushNotifEnabled — New setting (v2.1.119+) to allow Claude to send proactive push notifications via Remote Control when a long task finishes; default false
  • autoCompactEnabled — New setting (v2.1.119+) to automatically compact the conversation when context approaches the limit; default true
  • fileCheckpointingEnabled — New setting (v2.1.119+) to snapshot files before each edit for /rewind restore; default true
  • inputNeededNotifEnabled — New setting (v2.1.119+) to send a push notification when a permission prompt is waiting for input; default false
  • theme — New setting (v2.1.119+) to configure the interface color theme (auto, dark, light, daltonized, ANSI, or custom slug); default dark
  • verbose — New setting (v2.1.119+) to show full tool output instead of truncated summaries; default false
  • Legacy key note updated — ~/.claude.json note now lists theme, verbose, autoCompactEnabled, and preferredNotifChannel as keys stored there in versions before v2.1.119
code.claude.com/docs/en/settings →
Requires Claude Code + claude-docs plugin
