Key Highlights

  • mid-conversation-system-2026-04-07 beta flag — New AnthropicBeta value added across all API endpoints, expanding the enumerated beta flag set from 23 to 24 supported values.
  • BetaManagedAgentsModel roster change — claude-opus-4-8 removed from the BetaManagedAgentsModel enumeration across all Agents API endpoints (create, update, list, retrieve, archive, versions); claude-sonnet-4-6 now leads the remaining 6-model list.
  • Agents API (all SDKs &amp; REST) — All agent CRUD and version endpoints updated with the new beta flag and revised model list across Python, TypeScript, Go, Java, Ruby, C#, CLI, and REST variants.
  • Environments API (all SDKs &amp; REST) — All environment CRUD and Work sub-resource endpoints (poll, ack, heartbeat, stats, stop, update, list, retrieve) updated with the new beta flag across all SDKs.
  • Volume note — 262 API reference docs were updated in this batch; individual SDK variants (Python, TypeScript, Go, Java, Ruby, C#, PHP, CLI) each received the same flag and model changes, so per-SDK entries are omitted here for brevity.
  • mid-conversation-system-2026-04-07 beta flag — The new mid-conversation-system-2026-04-07 beta header value has been added to all beta API endpoints (Files, Memory Stores, Memories, Memory Versions) across REST, Python, TypeScript, Go, Java, Ruby, C#, and CLI SDKs, expanding the total supported beta flags from 23 to 24.
