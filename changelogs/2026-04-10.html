Release Notes

Key Highlights

  • advisor-tool-2026-03-01 — New beta flag added to the AnthropicBeta enum across all API endpoints and SDK clients (Python, TypeScript, Go, Ruby, Java, C#), expanding the supported beta version count from 18 to 19.
  • advisor tool — The Messages API (create, count tokens, batches) now exposes "advisor" as a new tool name alongside web_search, web_fetch, and code_execution, with a corresponding BetaAdvisorToolResultBlockParam response type.
  • Environments (create/archive/list/retrieve/update/delete) — Removed the sessionKey cookie parameter from all environment endpoints, replacing it with the new advisor-tool-2026-03-01 beta flag.
  • Sessions metadata — The metadata field on session create/update now documents its constraint: maximum 16 pairs, keys up to 64 chars, values up to 512 chars.
  • Vaults metadata — The metadata field on vault create now documents its constraint: maximum 16 pairs, keys up to 64 chars, values up to 512 chars.
  • Java SDK Agents — Three new docs added for the Java SDK: beta/agents, beta/agents/create, and beta/agents/update, bringing Java Agents coverage to parity with other SDKs.
