Key Highlights

  • UserPromptExpansion hook event — New hook event fires when a slash command expands into a prompt before reaching Claude; supports blocking, context injection, and matcher on command name across hooks reference, hooks guide, and plugins reference.
  • WebFetch domain safety check — New per-hostname blocklist preflight added to the WebFetch tool (runs against api.anthropic.com regardless of provider); documented in data-usage, settings (skipWebFetchPreflight), and network-config.
