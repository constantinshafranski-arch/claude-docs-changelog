Key Highlights

  • Managed Agents API (beta) — New beta endpoints for Agents (`/v1/agents`), Environments (`/v1/environments`), Sessions (`/v1/sessions`), and Vaults (`/v1/vaults`) cover the full lifecycle of managed agent execution, including CRUD, archiving, events, streaming, and credential storage; mirrored across Python, TypeScript, Go, Java, Ruby, C#, and CLI SDKs.
  • claude-mythos-preview model — New model added to the Messages API model union — described as a "new class of intelligence, strongest in coding and cybersecurity"; `claude-opus-4-6` description updated from "Most intelligent" to "Frontier intelligence for long-running agents and coding".
  • Files API scope field — `FileMetadata` response now includes an optional `scope` field (`{id, type: "session"}`) on upload, retrieve, and list endpoints; `List Files` gains a `scope_id` query parameter to filter by session.
  • Beta headers code examples — Shell (`curl`) and CLI (`ant`) examples moved to the top of the beta headers code group, making them more prominent alongside existing Python/TypeScript samples.
  • Beta API `user-profiles-2026-03-24` removed — The `user-profiles-2026-03-24` beta flag was removed from the `AnthropicBeta` union across all endpoints and SDKs (count dropped from 19 to 18 members).
  • 150+ new API reference docs — Agents, Environments, Sessions, Vaults, and CLI SDK pages were all newly added this release; many additional per-SDK variants (C#, Go, Java, Ruby, TypeScript) were added in parallel but are omitted here for brevity.
