Key Highlights

  • Managed Agents API (beta) — New beta endpoints for Agents, Environments, Sessions, Vaults, and Resources added across all SDKs (Python, TypeScript, Go, Ruby, Java, C#, CLI), enabling full lifecycle management of stateful AI agents.
  • Sessions API — New POST /v1/sessions endpoint with full CRUD, event streaming (GET /stream, POST /events), and session-scoped resource management for persistent agent execution contexts.
  • Files API scope field — FileMetadata response now includes an optional scope field ({ id, type: "session" }) indicating the session context a file was created in; GET /v1/files list accepts new scope_id filter.
  • claude-mythos-preview model — New claude-mythos-preview model added to the API (described as "new class of intelligence, strongest in coding and cybersecurity"); claude-opus-4-6 description updated to "Frontier intelligence for long-running agents and coding".
  • Beta header code examples — Shell and CLI code samples reordered in the Beta Headers doc so Shell and CLI examples appear before Python/TypeScript examples.
  • AnthropicBeta enum reduced — user-profiles-2026-03-24 beta flag removed from the AnthropicBeta union across all API and SDK reference docs, reducing the member count from 19 to 18.
