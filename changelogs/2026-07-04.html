Key Highlights

  • Google Vertex AI renamed to Google Cloud's Agent Platform — All documentation across 20+ pages updated to replace "Google Vertex AI" and "Vertex AI" references with "Google Cloud's Agent Platform", including setup guides, CI/CD integrations, and settings references
  • --cloud flag introduced — The --remote flag for creating web sessions is now deprecated in favor of --cloud; --remote remains as a deprecated alias
  • manual permission mode alias added (v2.1.200) — manual is now accepted as an alias for the default permission mode across --permission-mode, defaultMode settings, hooks setMode, subagent frontmatter, and the VS Code initialPermissionMode setting
  • modelOverrides now applies to --model and ANTHROPIC_MODEL (v2.1.200) — Anthropic model IDs passed via CLI flag or environment variable are now mapped through modelOverrides on Amazon Bedrock, Google Cloud's Agent Platform, and Mantle, instead of being forwarded as-is
  • askUserQuestionTimeout setting added (v2.1.200) — New setting controls auto-continue timeout for AskUserQuestion dialogs; accepts "60s", "5m", "10m", or "never"
  • Amazon Bedrock naming standardized — All references to plain "Bedrock" updated to "Amazon Bedrock" for consistency across authentication, IAM, and cost-tracking documentation
