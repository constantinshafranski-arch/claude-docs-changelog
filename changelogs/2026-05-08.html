The Beta API reference was extended with a new Webhooks section covering the full BetaWebhookEvent type hierarchy for session and vault lifecycle events.

New

Claude Code settings

Updated

Multiple new settings were added (parentSettingsBehavior, worktree.baseRef, claudeMdExcludes, syntaxHighlightingDisabled, bwrapPath, socatPath) and several existing entries were updated with additional env var cross-references and clarified descriptions.

  • parentSettingsBehavior — New managed-only setting (v2.1.133+): controls whether host-supplied managed settings (from Agent SDK or IDE extensions) apply alongside admin-deployed managed tiers; values are "first-wins" (default, drops parent settings) or "merge" (parent settings apply under admin tier, policy-tightening only).
  • worktree.baseRef — New worktree setting: "fresh" (default) branches from origin/&lt;default-branch&gt;; "head" branches from local HEAD, preserving unpushed commits. Applies to --worktree, EnterWorktree tool, and subagent isolation.
  • claudeMdExcludes — New setting: glob patterns or absolute paths of CLAUDE.md files to skip when loading memory; matches against absolute paths; cannot exclude managed policy files.
  • syntaxHighlightingDisabled — New setting: disables syntax highlighting in diffs, code blocks, and file previews.
  • bwrapPath / socatPath — New managed-only sandbox settings (Linux/WSL2): override automatic PATH-based detection for bwrap and socat binaries respectively.
  • gcpAuthRefresh — New setting: custom script to refresh GCP Application Default Credentials on expiry; documented with example and reference to Vertex AI advanced credential configuration.
  • alwaysThinkingEnabled — Added reference to CLAUDE_CODE_DISABLE_THINKING env var for forcing thinking off.
  • apiKeyHelper — Added reference to CLAUDE_CODE_API_KEY_HELPER_TTL_MS for configuring refresh interval.
  • autoMemoryEnabled — Added reference to CLAUDE_CODE_DISABLE_AUTO_MEMORY env var.
  • effortLevel — Added that --effort and CLAUDE_CODE_EFFORT_LEVEL override this for one session.
  • feedbackSurveyRate — Added reference to CLAUDE_CODE_DISABLE_FEEDBACK_SURVEY env var as an alternative suppression method.
  • model — Clarified that --model and ANTHROPIC_MODEL override this for one session.
  • otelHeadersHelper — Added reference to CLAUDE_CODE_OTEL_HEADERS_HELPER_DEBOUNCE_MS for configuring the refresh debounce interval.
