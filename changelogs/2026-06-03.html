Key Highlights

  • New model: claude-opus-4-8 — Added claude-opus-4-8 as a new frontier model for long-running agents and coding across all Agents API endpoints and multi-language SDK references.
  • Beta flag removed: mid-conversation-system-2026-04-07 — The mid-conversation-system-2026-04-07 beta flag has been removed from the AnthropicBeta enum across all 278 updated API reference docs, reducing the enumerated count from 24 to 23 additional values.
  • Massive cross-language API refresh — All 278 API reference docs were updated simultaneously across REST, Python, TypeScript, Go, Java, Ruby, C#, CLI, and PHP SDKs to reflect the model and beta flag changes.
  • Agents endpoints updated — All Create, Update, List, Get, Archive, and Versions endpoints for /v1/agents now surface claude-opus-4-8 as the leading model option in BetaManagedAgentsModel.
  • Self-hosted environment work endpoints updated — All work queue endpoints (poll, ack, heartbeat, stop, update, list, stats, retrieve) reflect the removal of the mid-conversation-system-2026-04-07 beta header value.
  • Memory Stores, Files, and other beta resources updated — Endpoints for memory stores, memories, memory versions, files, environments, sessions, skills, vaults, user profiles, and credentials were all refreshed in this commit. Many additional entries beyond those shown were omitted due to volume.
